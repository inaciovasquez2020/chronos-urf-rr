# Newstein Fundamental-Cycle-to-Local-Coboundary Promotion

## Status
OPEN

## Statement
Assume the Newstein Fundamental Cycle Generation Sublemma.

Let \(\omega\) be a \(1\)-cocycle on \(B_R(v)\) supported on the rooted-local cycle sector.
If \(\omega\) has zero evaluation on every rooted-local cycle generator and vanishes on every rooted boundary class entering the quotient, then \(\omega\) is a coboundary on \(B_R(v)\).

## Role
This packages the final promotion step into the Newstein Local Coboundary Criterion.

## Consequence
Once the fundamental-cycle generation step is proved, the Local Coboundary Criterion reduces to the existing spanning-forest proof blueprint.
