# Newstein Global Direct-Sum Embedding Corollary

Status: OPEN

## Statement

Assume the Fiber-to-Global Injectivity Diameter Bridge:
\[
u\neq v,\qquad
\iota_u([c_u])=\iota_v([c_v])
\Longrightarrow
[c_u]=0=[c_v].
\]

Assume each nonzero fiber quotient
\[
Z_1(B_r(u);\mathbf F_2)/W_u
\]
contains the \(2\)-dimensional twisted contribution furnished by the sigma-package.

Then the induced map
\[
\bigoplus_{u}\bigl(Z_1(B_r(u);\mathbf F_2)/W_u\bigr)
\longrightarrow
Z_1/W^{\mathrm{glob}}
\]
is injective on distinct fiber summands, and the global quotient contains the corresponding direct sum of local \(2\)-dimensional twisted pieces.

## Role

This is the exact corollary turning pairwise injectivity into a global direct-sum lower bound.
