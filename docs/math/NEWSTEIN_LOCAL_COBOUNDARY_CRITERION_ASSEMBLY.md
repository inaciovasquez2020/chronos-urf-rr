# Newstein Local Coboundary Criterion Assembly

## Status
OPEN

## Statement
Let \(T\) be a rooted spanning tree of a graph \(G\). Let \(B\) be a rooted ball in \(G\). If every non-tree edge of \(B\) generates a fundamental cycle entirely contained in \(B\), then the cycle contribution on \(B\) is generated by rooted-local fundamental cycles. Consequently, the obstruction class on \(B\) is detected by the local fundamental-cycle span, yielding the local coboundary criterion.

## Assembly inputs
1. Fundamental cycle generation assembly.
2. Restriction of rooted spanning-tree data to a rooted ball.
3. Linear generation of local cycle space by fundamental cycles of non-tree edges.

## Proof skeleton
1. By fundamental cycle generation assembly, each non-tree edge \(e\) in \(B\) generates a unique fundamental cycle \(C_T(e)\).
2. By hypothesis, each such cycle is entirely contained in \(B\).
3. Therefore the local cycle space of \(B\) is generated by rooted-local fundamental cycles coming from non-tree edges of \(B\).
4. Any obstruction class on \(B\) vanishes exactly when it vanishes on this generating family.
5. Hence the local coboundary criterion holds.

\[
\mathrm{LocalCoboundaryCriterion}
