# Newstein Fundamental Cycle Generation Proof Blueprint

## Status
OPEN

## Target
Prove the Newstein Fundamental Cycle Generation Sublemma.

## Input
Let \(B_R(v)\) be a rooted radius-\(R\) ball in a bounded-degree graph.
Let
\[
T \subseteq B_R(v)
\]
be a rooted spanning forest.
Let
\[
e = \{x,y\} \in E(B_R(v)) \setminus E(T)
\]
be a non-tree edge.

## Required output
Show that the unique fundamental cycle
\[
C_e \subseteq T \cup \{e\}
\]
lies in the rooted-local generating family used by the Newstein quotient.

## Minimal proof decomposition
1. Prove uniqueness of the tree path \(P_T(x,y)\) in \(T\).
2. Define the fundamental cycle
\[
C_e := P_T(x,y) \cup \{e\}.
\]
3. Show that \(C_e\) is contained in \(B_R(v)\).
4. Show that \(C_e\) is rooted-local in the sense used by the Newstein quotient.
5. Identify the generating family in the quotient definition.
6. Prove that every rooted-local fundamental cycle is a member of, or generated by, that family.
7. Conclude that every non-tree edge contribution is testable against Newstein rooted-local cycle generators.

## Weakest missing sublemma
Every tree path \(P_T(x,y)\) arising from a non-tree edge in \(B_R(v)\) remains inside the rooted-local radius regime defining the Newstein generating family.

## Finish condition
This file is replaceable by a proof exactly when the weakest missing sublemma is discharged.
