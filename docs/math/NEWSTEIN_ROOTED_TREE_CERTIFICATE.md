# NEWSTEIN_ROOTED_TREE_CERTIFICATE

Status: OPEN

## Statement

Let (B_R(r)) be a rooted ball with parent map (p) defined on (B_R(r)\setminus{r}).
Assume the parent-depth-length identity holds:
[
\forall v\in B_R(r)\setminus{r},\quad d\bigl(r,p(v)\bigr)=d(r,v)-1.
]

Then the parent map is a strict descent operator:
[
\forall v\in B_R(r)\setminus{r},\quad d\bigl(r,p(v)\bigr)<d(r,v).
]

## Role in the closure chain

This is the rooted-tree certificate needed to force parent iteration toward the root.
It is the structural bridge from the parent map to local acyclicity and the local coboundary criterion.
