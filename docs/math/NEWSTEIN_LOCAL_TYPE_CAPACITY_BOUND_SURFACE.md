# Newstein Local-Type Capacity Bound Surface

Status: OPEN

## Statement

Assume the quotient invariant \(Q\) factors through a bounded local type space depending only on \((r,k,\Delta)\).

The theorem-level target is:
\[
\dim_{\mathbf F_2}(Z_1/W^{\mathrm{glob}})\le C(r,k,\Delta)
\]
for some constant \(C(r,k,\Delta)\) independent of the graph size.

In particular, bounded local-type factorization forbids unbounded sigma-growth in the global quotient.

## Role

This is the exact theorem-level proof surface for the bounded-capacity half of the final contradiction.
