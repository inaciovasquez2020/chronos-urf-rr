# Newstein Rooted-Local Tree-Path Lemma

Status: OPEN

## Statement

Let \(L\) be a connected rooted local complex and let \(T\subseteq L\) be a spanning tree. Then for every non-tree edge \(e=uv\in E(L)\setminus E(T)\), the unique tree path \(P_T(u,v)\subseteq T\) is rooted-local in \(L\).
