# Newstein Direct-Sum Independence Lemma

## Status
OPEN

## Statement
Let \(\{Q(F_i)\}_{i \in I}\) be the residual fiber quotients in the Newstein decomposition.
Assume each \(Q(F_i)\) is realized by a supported fiber class.

Then no nontrivial finite linear combination of classes supported on pairwise distinct fibers vanishes in the ambient direct-sum realization.

Equivalently, for every finite subset \(J \subseteq I\), the canonical map
\[
\bigoplus_{i \in J} Q(F_i) \longrightarrow \sum_{i \in J} Q(F_i)
\]
is injective.

## Role
This excludes hidden cross-fiber cancellation before passage to the global quotient.
