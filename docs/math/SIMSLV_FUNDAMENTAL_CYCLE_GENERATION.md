# SiMSLV Fundamental Cycle Generation

Status: OPEN

## Statement

Fix \(r\), a rooted vertex \(x\), and the rooted radius-\(r\) ball \(B_r(x)\) in the witness-layer \(2\)-complex.

Assume:

\[
L>2r+1,
\qquad
\operatorname{girth}(X_n)>2r+1,
\]
and assume a locality-preserving triangle predicate
\[
\Phi_2(u,v,w)
\]
has been specified on \(B_r(x)\).

Then
\[
Z_1(B_r(x);\mathbb F_2)
=
\left\langle \partial\tau :
\tau=[u,v,w]\subseteq B_r(x),\ \Phi_2(u,v,w)
\right\rangle_{\mathbb F_2}.
\]

## Interpretation

Every local \(\mathbb F_2\)-cycle in the rooted radius-\(r\) ball is generated by admissible local triangle boundaries.

## Role

This is the weakest sufficient continuation object for the rooted-ball branch. Combined with triangle-vanishing, it yields local cycle-vanishing and then rooted-ball trivialization.

Replace this OPEN statement by a theorem-level proof.
