# NEWSTEIN_LOCAL_ACYCLICITY

Status: OPEN

## Statement

Let (B_R(r)) be a rooted ball with parent map (p) defined on (B_R(r)\setminus{r}).
Assume the rooted-tree certificate holds:
[
\forall v\in B_R(r)\setminus{r},\quad d\bigl(r,p(v)\bigr)<d(r,v).
]

Then (B_R(r)) is locally acyclic in degree (1):
[
H_1\bigl(B_R(r);\mathbf F_2\bigr)=0.
]

## Role in the closure chain

This is the exact homological bridge from strict parent descent to the local coboundary criterion.
It is the immediate predecessor to rebuilding `NEWSTEIN_LOCAL_COBOUNDARY_CRITERION.md`.
