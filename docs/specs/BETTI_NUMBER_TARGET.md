# Betti Number Target

Open problem.

Required unconditional Lean target:

\[
\beta_1(G)=\dim_{\mathbf F_2} Z_1(G)=|E|-|V|+c(G).
\]

Missing ingredients:

1. formal definition of connected components \(c(G)\),
2. rank-nullity for the boundary operator \(\partial_1\),
3. identification of \(\operatorname{im}(\partial_1)\) with the cut space,
4. dimension formula for the cut space.

No placeholder Lean theorem should be committed for this statement until these ingredients are formalized.
