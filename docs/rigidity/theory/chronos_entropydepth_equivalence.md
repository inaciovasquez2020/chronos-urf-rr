# Chronos–EntropyDepth Equivalence

## Statement

Let G be a bounded-degree graph family and let A be any FO^k-local refinement system.

Define the EntropyDepth ED(G) as the minimum number of refinement steps required to reduce the configuration entropy to zero.

Then the Chronos wall condition

t ≥ Ω(|V(G)|)

is equivalent to the statement

ED(G) ≥ Ω(|V(G)|).

## Interpretation

The Chronos wall is the algorithmic manifestation of the EntropyDepth invariant.

Any system whose refinement depth is bounded below by a linear function of the input size exhibits the Chronos barrier.

## Structural Role

EntropyDepth provides the invariant formulation of the Chronos obstruction.

Chronos Wall  
⇔ EntropyDepth ≥ Ω(n).
