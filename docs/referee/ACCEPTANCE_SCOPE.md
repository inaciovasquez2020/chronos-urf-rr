Acceptance Scope — Chronos / EntropyDepth

This repository is designed to be review-safe.

• All unconditional claims are isolated and independently verifiable.
• All stronger results are explicitly conditional with named hypotheses.
• No Clay Millennium Problem is claimed solved.
• No empirical or heuristic argument is used as proof.
• All claims are reproducible from the repository or stated dependencies.

The intended contribution is structural and foundational.
