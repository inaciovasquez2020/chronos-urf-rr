Cycle Packing Bound

Let cycles be pairwise edge-disjoint.

If each cycle has length ≤ L then

|UnionEdges| ≥ Σ cycle_length

Thus

|UnionEdges| ≥ |cycles|

and

|cycles| ≤ |UnionEdges| / L

For bounded-degree graphs

|E| = d n / 2

Thus

packing ≥ (|E| − |V| + 1)/L
