# Oblivion Atom — Torus Counterexample

## Status
**Original Oblivion Atom statement: REFUTED**

This document records the explicit counterexample showing that the original
cycle–overlap rigidity hypothesis (the "Oblivion Atom") does not hold in full
generality.

---

# 1. Original Statement (Oblivion Atom)

For fixed parameters \(k\) and \(\Delta\):

\[
\forall k,\Delta\;\exists R,T\;\forall G:
\deg(G)\le\Delta \land |\{\tau_R^k(v)\}|=1
\Rightarrow C_R(G)\le T
\]

where

* \( \tau_R^k(v) \) = FO\(^k\) local type of the radius-\(R\) neighborhood of \(v\)
* \( C_R(G) = \dim_{\mathbb{F}_2}\operatorname{Im}(\Phi_R) \)

and

\[
\Phi_R :
\bigoplus_{v\in V(G)} Z(B_R(v))
\rightarrow
Z(G)
\]

maps local cycle spaces into the global cycle space.

Interpretation:

If all vertices have identical FO\(^k\) neighborhoods, the cycle-overlap rank
generated by local neighborhoods should remain bounded.

---

# 2. Counterexample Family

Define the graph family

\[
G_n = \operatorname{Cay}(\mathbb Z_n \times \mathbb Z_n,
\{(\pm1,0),(0,\pm1)\})
\]

This is the **discrete 2-torus grid**.

Properties:

* vertex count: \( |V(G_n)| = n^2 \)
* degree: \( \deg(G_n) = 4 \)
* edges: \( |E(G_n)| = 2n^2 \)

---

# 3. FO^k Homogeneity

Translations

\[
\phi_g(x)=x+g
\]

are graph automorphisms.

Thus

\[
(B_R(u),u) \cong (B_R(v),v)
\]

for all \(u,v\).

Therefore

\[
|\{\tau_R^k(v)\}| = 1
\]

for every \(k\) and every fixed \(R\) provided \(n>2R\).

---

# 4. Local Cycle Structure

For \(R\ge2\), each radius-\(R\) ball contains **unit squares**:

v ---- v
| |
v ---- v


These 4-cycles generate the local cycle space.

The number of independent cycle **shapes** inside a single ball is constant:

\[
\beta_1(B_R(v)) = O(R^2)
\]

---

# 5. Global Cycle Generation

Each vertex corresponds to a translated unit square.

There are \(n^2\) such squares.

Their edge-incidence vectors over \(\mathbb{F}_2\) satisfy exactly one relation:

\[
\bigoplus_{\square} \mathbf{1}_\square = 0
\]

Thus their span has dimension

\[
n^2 - 1
\]

This equals the cycle space rank:

\[
\beta_1(G_n)
=
|E| - |V| + 1
=
2n^2 - n^2 + 1 - 1
=
n^2 - 1
\]

---

# 6. Cycle-Overlap Rank

Since every unit square lies inside some radius-\(R\) ball,

\[
C_R(G_n)
=
\dim_{\mathbb F_2} \operatorname{Im}(\Phi_R)
=
n^2 - 1
\]

Hence

\[
C_R(G_n) \rightarrow \infty
\]

as \(n \rightarrow \infty\).

---

# 7. Failure of the Oblivion Atom

We now check the conditions of the statement.

| Condition | Torus |
|-----------|------|
| bounded degree | ✓ |
| FO\(^k\) homogeneity | ✓ |
| bounded \(C_R(G)\) | ✗ |

Because

\[
C_R(G_n) = n^2 - 1
\]

the implication fails.

Thus no constants \(R,T\) can satisfy the Oblivion Atom universally.

---

# 8. Failure Mechanism

The key phenomenon is **translation replication**.

Although the number of cycle shapes inside a ball is bounded,

their **translations across the graph are linearly independent**.

Thus

bounded local cycle types
≠
bounded global span


---

# 9. Implication for Chronos / EntropyDepth

The invariant

\[
C_R(G)
\]

cannot serve as a universal rigidity measure.

Local homogeneity does **not** prevent linear growth of the cycle span.

The Chronos program therefore requires a modified invariant.
