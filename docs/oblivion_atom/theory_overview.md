# Oblivion Atom — Theory Overview

Goal

Explain how local cycle structure in bounded-degree graphs forces
logical distinguishability under FO^k formulas.

Core Mechanism

Cycle-overlap growth creates structural asymmetries in
radius-R neighborhoods.

These asymmetries propagate into FO^k neighborhood types,
preventing global logical homogeneity.

Pipeline

Local cycles
→ cycle-overlap rank
→ patch dependency chains
→ FO^k type divergence.

Diagnostics Implemented

Geometry signature
Cycle rank probes
Patch dependency analysis
FO^k collision detector

Target Result

COR_R(G) ≥ T ⇒ G not FO^k_R-homogeneous.

Interpretation

Logical indistinguishability cannot survive sufficiently
rich overlapping cycle structure in bounded-degree graphs.
