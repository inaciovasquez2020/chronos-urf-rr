Cycle-Overlap Rank (COR)

COR_R(G) is defined as the rank of the patch-cycle incidence matrix.

Rows = cycles
Columns = R-patches

Entry(i,j) = 1 if cycle_i ⊂ patch_j

Thus

COR_R(G) = rank_F2(M)

This equals the dimension of the projection of the cycle space onto patch supports.
