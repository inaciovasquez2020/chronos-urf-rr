# Cycle–Type Exposure Lemma (Open)

## Statement

Fix k, Δ.

There exists c = c(k,Δ) such that for every finite graph G with
max degree ≤ Δ and radius R,

COR_R(G) > c

implies the existence of vertices u,v with

tp^FO^k_R(u) ≠ tp^FO^k_R(v).

Here COR_R(G) = dim_F2 Im(Φ_R) where

Φ_R : ⊕_v Z(B_R(v)) → Z(G)

is the cycle-overlap map.

## Role in the Oblivion Atom

This lemma is the missing bridge:

cycle-overlap rank  ⇒  FO^k local type diversity

Without it the Oblivion Atom statement cannot be proved.
