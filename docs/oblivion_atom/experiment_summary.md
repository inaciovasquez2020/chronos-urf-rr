# Oblivion Atom Experiments — Summary

## Objective

Empirically study the relationship between cycle-incidence rank and local logical homogeneity.

---

## Experiment Classes

### Random Regular Graphs

Observation:

rank_F2(M_R(G)) ≈ Θ(n)

Vertex WL orbits ≈ n

Conclusion:

Random regular graphs are not FO^k-homogeneous.

---

### Random Lifts

Observation:

rank_F2(M_R(G)) ≈ Θ(n)

Moderate orbit collapse but still linear rank.

Conclusion:

Local diversity persists.

---

### CFI Homogeneous Proxy

Observation:

vertex orbit count = 1

cycle orbit count = 2

compressed rank = 0

Conclusion:

Homogeneous graphs collapse cycle-incidence rank.

---

## Structural Insight

Cycle-incidence rank separates two regimes:

Non-homogeneous graphs → linear rank

Homogeneous graphs → bounded rank

---

## Implication

This supports the Cycle–Overlap Rank Rigidity hypothesis.
