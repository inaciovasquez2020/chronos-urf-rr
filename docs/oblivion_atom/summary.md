# Oblivion Atom — Module Summary

The Oblivion Atom module investigates how overlapping cycle structure
in bounded-degree graphs forces logical distinguishability under FO^k.

## Implemented Layers

Geometry diagnostics  
Cycle-overlap diagnostics  
FO^k logical collision detection  

## Core Observable

(|B_R|, β₁(B_R), ρ(R))

where

B_R is the radius-R ball  
β₁(B_R) is cycle rank  
ρ(R) = β₁(B_R)/|B_R|
