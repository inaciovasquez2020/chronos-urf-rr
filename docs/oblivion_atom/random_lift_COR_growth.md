# Random Lift COR Growth

Let H be a base graph containing a cycle.

Construct random n-lift G_n.

Properties:

|V(G_n)| = n|V(H)|

Edges lifted via random permutations.

Result.

With high probability,

COR_R(G_n) = Ω(n)

Reason.

Independent lifts of base cycles produce Θ(n) disjoint or weakly overlapping cycles.

Thus the cycle-overlap rank grows linearly with graph size.
