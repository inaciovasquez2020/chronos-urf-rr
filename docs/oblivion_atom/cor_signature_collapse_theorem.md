# COR Signature Collapse Theorem

Let G be a bounded-degree graph with degree ≤ Δ.

Let R be fixed.

Define COR_R(G) as the cycle-overlap rank.

If

COR_R(G) > C(k,Δ,R)

then there exist vertices u,v such that

(G,u) ≠_{FO^k_R} (G,v).

Reason:

Large COR_R(G) forces many distinct cycle participation signatures.
FO^k_R homogeneity allows only finitely many signatures.
Thus signatures must differ for some vertices.
