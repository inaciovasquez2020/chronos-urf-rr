# Oblivion Atom — Core Statement

Fix k, Δ ∈ ℕ.

There exist constants R = R(k,Δ) and T = T(k,Δ) such that for every finite
graph G with maximum degree ≤ Δ:

COR_R(G) ≥ T  ⇒  G is not FO^k_R-homogeneous.

Definitions

COR_R(G)

Cycle-overlap rank generated by all cycles contained in radius-R balls.

FO^k_R-homogeneous

All vertices have the same FO^k neighborhood type up to radius R.

Interpretation

Large overlapping cycle structure forces logical distinguishability
between vertices under FO^k formulas.

Program Role

This lemma forms the final structural bridge between

cycle-overlap combinatorics
and
logical type diversity

in the Chronos / Oblivion framework.
