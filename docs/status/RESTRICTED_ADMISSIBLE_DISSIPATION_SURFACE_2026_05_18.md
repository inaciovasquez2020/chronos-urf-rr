# Restricted Admissible Dissipation Surface

Status: `INTERFACE_BRIDGE_ONLY`

This file records that a restricted entropy-dissipation certificate is already a restricted admissible-dissipation surface when the entropy functional is used as the dissipation functional.

It also records two inherited routes:

- uniform entropy control restricts to restricted admissible dissipation
- same-functional uniform Lyapunov decay restricts to restricted admissible dissipation

Boundary:

Bridge only.

Does not prove:

- existence of admissible dissipation certificates
- construction of an admissible domain
- entropy production
- entropy monotonicity for arbitrary entropy functions
- unrestricted admissible dissipation
- unrestricted `UniversalFiberEntropyGap`
- unrestricted Chronos-RR
- unrestricted H4.1/FGL
- P vs NP
- any Clay problem
