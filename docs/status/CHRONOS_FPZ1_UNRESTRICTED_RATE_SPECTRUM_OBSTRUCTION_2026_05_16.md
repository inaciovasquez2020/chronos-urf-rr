# Chronos FPz1 unrestricted rate-spectrum obstruction

Status: UNRESTRICTED_RATE_SPECTRUM_OBSTRUCTION_ONLY

## Formalized obstruction

The Lean module proves:

- `inverseNatRateSequence_refutes_rateSpectrumIsolation`

If the unrestricted admissible domain contains witnesses with rank-rate equal to `1 / n` for every `n >= 1`, then unrestricted `RateSpectrumIsolation` is false.
