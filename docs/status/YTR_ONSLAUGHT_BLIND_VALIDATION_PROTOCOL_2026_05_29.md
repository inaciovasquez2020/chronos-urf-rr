## Closed repository-native object

This object records the promotion rule for a candidate predictive dark-matter/GDM validation target.

A YtR result is not admissibly called new physics unless both gates pass:

1. blind likelihood improvement;
2. independent replication.

- YtR cannot be promoted to new physics before blind likelihood execution;
- YtR cannot be promoted to new physics before independent replication;
- new-physics promotion requires both gates.
