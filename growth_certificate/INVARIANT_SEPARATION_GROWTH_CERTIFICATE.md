# Invariant Separation Growth Certificate

## Target
Certify quantitative growth of the URF invariant separation for the witness family.

## Required Components
1. witness-family input interface
2. invariant evaluation interface
3. lower-bound statement format
4. asymptotic growth registry
5. finite certificate format

## Core Statement
There exists c > 0 such that for all sufficiently large n:
I_URF(G_n^-) - I_URF(G_n^+) ≥ c n

## Blocking Gap
Need explicit witness-family instances and executable invariant computation on those instances.
