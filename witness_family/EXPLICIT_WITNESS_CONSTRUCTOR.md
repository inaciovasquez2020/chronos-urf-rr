# Explicit Witness Constructor

## Target
Construct a finite witness family W(n) = (G_n^+, G_n^-).

## Required Properties
1. bounded degree
2. FO^k local indistinguishability at radius R
3. URF invariant separation
4. quantitative growth lower bound

## Constructor Schema
Input:
- base graph B_n
- lift assignment sigma_n : E(B_n) -> F2

Output:
- G_n^+ : untwisted lift
- G_n^- : twisted lift

## Blocking Gap
Need explicit finite base family with certified girth and executable lift encoding.
