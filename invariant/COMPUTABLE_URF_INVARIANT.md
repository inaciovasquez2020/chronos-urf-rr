# Computable URF Invariant

## Target
Define I_URF(G;R) = dim_F2(Z1(G) / Z1^{≤ 2R+1}(G)) in executable form.

## Required Components
1. edge encoding over F2
2. cycle-space kernel definition
3. local-cycle-span definition
4. quotient-rank computation
5. finite certificate format

## Blocking Gap
Need fixed graph encoding and explicit cycle enumeration interface for finite graphs.
