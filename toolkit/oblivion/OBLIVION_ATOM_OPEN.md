# Oblivion Atom (Open)

## Statement
Fix k, Δ ∈ ℕ. There exist R = R(k,Δ) and T = T(k,Δ) such that for every finite graph G with max degree ≤ Δ,
if COR_R(G) ≥ T then G is not FO^k_R-homogeneous.

## Status
OPEN: missing rigidity lemma (cycle-overlap rank ⇒ local type diversity).

## Equivalent forms
- Cycle-support rigidity
- Hypergraph gadget rigidity
- FO^k configuration pumping on bounded-degree graphs
