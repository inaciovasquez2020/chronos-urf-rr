# Vardi Observation — Bounded Treewidth Collapse

## Statement
For graphs G with bounded treewidth:
Z1(G) is generated by local cycles.

## Formal
∀ t, ∃ R:
tw(G) ≤ t ⇒ Z1(G) = ⟨ Z1^{≤R}(G) ⟩

## Consequence
Local → Global factorization holds in bounded-treewidth regime.

## Implication for URF
URF invariant collapses on bounded-treewidth graphs.

## Required Restriction
Non-factorization must be established on:
{ G : tw(G) → ∞ }
