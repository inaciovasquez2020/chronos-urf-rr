# Treewidth Guard Formalization

## Guard Schema
For each fixed t, there exists C_t such that
tw(G) ≤ t ⇒ I_URF(G) ≤ C_t.

## Role
This isolates the admissible regime for non-factorization:
tw(G) → ∞.

## Proof Obligation
Show that any bounded-treewidth decomposition induces generation of Z1(G)
from bounded-radius bag-supported cycles.

## Status
Required lemma layer before final unrestricted non-factorization packaging.
