\section{Alignment with EntropyDepth}

\begin{theorem}[Width–Entropy Alignment]
Let $F_n$ be a Tseitin family on constant-degree expanders.
Then any refinement process respecting resolution locality
must incur EntropyDepth
\[
ED(F_n) \ge c n
\]
for some constant $c>0$.
\end{theorem}
