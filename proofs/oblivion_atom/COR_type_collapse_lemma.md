# COR Type Collapse Lemma

Let G be a bounded-degree graph with degree ≤ Δ.

Assume FO^k_R homogeneity.

Then every radius-R neighborhood is logically identical.

Let B* denote the template ball.

Observation.

All s-local cycles arise from embeddings of cycles in B*.

Since Z_1(B*) has dimension ≤ O(Δ^R), the number of distinct cycle-support templates is bounded.

Thus

dim span{s-local cycles} ≤ T(k,Δ,R)·O(Δ^R).

Hence

COR_R(G) ≤ C(k,Δ,R).
