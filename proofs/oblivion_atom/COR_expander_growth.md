# Theorem: COR Linear Growth in Random Lifts

Let H be a base graph containing a cycle.

Let G_n be an n-lift of H.

Then with high probability

COR_R(G_n) = Ω(n).

Reason.

Each lift of the base cycle generates n lifted cycles.

Because overlap radius is bounded, a linear subset remains independent.

Therefore

∃ c(H,R) > 0 such that

COR_R(G_n) ≥ c(H,R)·n.
