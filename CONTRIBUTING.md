# Contributing to Chronos–URF RR

## Rules
- RR_INVARIANTS.md must remain valid.
- No semantic changes without invariant justification.
- CI must remain green.

## Allowed changes
- Documentation clarifications
- Toolchain and CI robustness
- Performance-neutral refactors

## Disallowed changes
- Changing Chronos semantics
- Bypassing URF Spine normalization
